Linear Mixed Model Analysis

  • LMM (Linear Mixed Model, 선형혼합모형)은 전체 집단에서 공통적으로 나타나는 fixed effects와 피험자마다 달라지는 random effects를 하나의 모형에서 함께 추정합니다.
  • fNIRS, DCS 및 speckle 기반 혈류 연구에서는 동일 피험자에서 반복 측정된 task, channel, measurement day 또는 장비 조건 사이의 상관을 고려하기 위해 LMM을 사용합니다.
  • 본 분석에서는 BFI와 BVI의 N-back load, SDS 및 processing 효과를 검정하면서 피험자별 평균 수준과 load-response 차이를 random effects로 반영합니다.

Savitzky–Golay Smoothing & Normalization

Savitzky–Golay Smoothing

Local Polynomial Method

\[ \widehat{\mathbf{a}}_i = \underset{\mathbf{a}}{\operatorname{arg\,min}} \sum_{j=-m}^{m} \left[ y_{i+j} - \sum_{r=0}^{p} a_r j^r \right]^2, \qquad \widetilde{y}_i=\widehat{a}_{i,0} \]
  • Sampling rate: \(200\ \mathrm{Hz}\)
  • Window length: \(30\ \mathrm{sec}\)
  • Polynomial order: \(3\)
  • Boundary mode: interp

Processing Pipeline

1 Data Preparation BFI/BVI Excel 신호와 Segments.xlsx를 불러오고, 비수치·결측값을 제거한 뒤 time 순으로 정렬합니다. 전체 시계열에 Savitzky–Golay smoothing을 적용하고 baseline 및 1–4 back 구간을 연결합니다.
2 Window Mean 각 baseline 및 N-back segment에서 중앙의 최대 120초를 선택합니다. 구간이 120초보다 짧으면 전체 구간을 사용하고, 선택된 sample의 평균을 해당 조건의 대표값으로 계산합니다.
3 Normalization BFI는 task mean과 baseline mean의 log ratio로 변환하고, BVI는 task mean에서 baseline mean을 뺀 difference로 변환하여 피험자별 절대 신호 수준 차이를 줄입니다.
4 Model-Input Construction 관측값을 Subject × Processing × SDS × Load의 long-format 자료로 구성합니다. Load 1–4는 평균 2.5를 기준으로 중심화하여 Load_c = Load − 2.5로 정의합니다.
5 LMM Fitting Primary model은 NoiseReduced 자료의 Load × SDS를 검정하고, Secondary model은 RawDarkOnly와 NoiseReduced 자료의 Load × Processing을 SDS 보정과 함께 검정합니다.
6 Inference and Export fixed effects, linear contrasts, Wald z test, likelihood-ratio test, random-effect variance와 ICC를 계산합니다. 적합 정보와 기술통계를 BFI/BVI별 Excel 결과 파일에 저장합니다.

Outcome Normalization

BFI Log-Ratio Outcome

\[ Y_{\mathrm{BFI}} = \ln \left( \frac{ \overline{\mathrm{BFI}}_{\mathrm{task}} }{ \overline{\mathrm{BFI}}_{\mathrm{baseline}} } \right) \]
  • \(Y=0\): baseline과 동일
  • \(Y>0\): task BFI 증가
  • \(Y<0\): task BFI 감소
  • baseline 또는 task mean이 \(10^{-12}\) 이하이면 제외

BVI Baseline-Difference Outcome

\[ Y_{\mathrm{BVI}} = \overline{\mathrm{BVI}}_{\mathrm{task}} - \overline{\mathrm{BVI}}_{\mathrm{baseline}} \]
  • \(Y=0\): baseline과 동일
  • \(Y>0\): task BVI 증가
  • \(Y<0\): task BVI 감소
  • BFI와 달리 logarithmic ratio를 추가로 적용하지 않음

Primary LMM: Load × SDS

Noise-Reduced Data Only

Model formula
\[ \mathrm{Outcome} \sim \mathrm{Load}_c * \mathrm{SDS} \]
\[ Y_{ijs} = \beta_0 +\beta_1L_{ij} +\beta_2S_s +\beta_3L_{ij}S_s +b_{0i} +b_{1i}L_{ij} +\varepsilon_{ijs} \]

Primary Question

  • NoiseReduced 데이터에서 load에 따른 변화가 존재하는가?
  • 15 mm와 20 mm의 load slope가 서로 다른가?
  • 평균 load에서 SDS 간 반응 차이가 존재하는가?
Fixed-Effect Term Statistical Meaning Interpretation
Intercept \(\beta_0\) 15 mm, load 2.5에서 예상되는 baseline-normalized Outcome
Load_c \(\beta_1\) 15 mm에서 load가 한 단계 증가할 때의 변화
C(SDS)[T.20mm] \(\beta_2\) 평균 load에서 20 mm − 15 mm 차이
Load_c:C(SDS)[T.20mm] \(\beta_3\) 20 mm slope − 15 mm slope

Primary Linear Contrasts

Load Slopes

\[ \mathrm{Slope}_{15\,mm}=\beta_1 \]
\[ \mathrm{Slope}_{20\,mm}=\beta_1+\beta_3 \]

Between-SDS Contrasts

\[ \Delta\mathrm{Slope}_{20-15}=\beta_3 \]
\[ \Delta\mathrm{SDS}_{20-15}\big|_{L_c=0}=\beta_2 \]

Secondary LMM: Load × Processing

RawDarkOnly and NoiseReduced

Model formula
\[ \mathrm{Outcome} \sim \mathrm{Load}_c * \mathrm{Processing} + \mathrm{SDS} \]
\[ Y = \beta_0+\beta_1L+\beta_2P +\beta_3LP+\beta_4S +b_0+b_1L+\varepsilon \]

Secondary Question

  • RawDarkOnly에서 load slope가 존재하는가?
  • NoiseReduced에서 load slope가 존재하는가?
  • 두 processing의 load slope가 서로 다른가?
  • SDS를 보정한 processing 평균 차이가 존재하는가?
Fixed-Effect Term Interpretation
Intercept RawDarkOnly, 15 mm, load 2.5에서 예상되는 Outcome
Load_c RawDarkOnly에서 load가 한 단계 증가할 때의 변화
C(Processing)[T.NoiseReduced] 평균 load에서 NoiseReduced − RawDarkOnly 차이
Load_c:C(Processing)[T.NoiseReduced] NoiseReduced slope − RawDarkOnly slope
C(SDS)[T.20mm] Load와 Processing을 보정한 평균적인 20 mm − 15 mm 차이
이 Secondary model에는 Processing × SDS, Load × SDS 및 Load × Processing × SDS interaction이 없습니다. 따라서 noise-reduction 효과가 15 mm와 20 mm에서 서로 다른지는 이 모델만으로 직접 검정하지 않습니다.

Random Effects and Model Fitting

Initial Random Structure

\[ (1+\mathrm{Load}_c\mid\mathrm{Subject}) \]
  • 피험자별 random intercept
  • 피험자별 random load slope
  • intercept–slope covariance

Fallback Structure

\[ (1\mid\mathrm{Subject}) \]
  • random-slope model 실패 시 적용
  • 피험자별 평균 수준 차이만 모델링
  • 최소 2개 이상의 load를 가진 subject만 유지

Optimization

  • Estimation: Maximum Likelihood
  • REML: False
  • Optimizers: L-BFGS, Powell, CG
  • Maximum iterations: 2000
  • 수렴 시도와 warning을 Excel에 기록

Statistical Inference

Wald z Test

\[ z=\frac{\widehat{\beta}}{SE(\widehat{\beta})} \]
\[ p=2\Phi(-|z|) \]
\[ CI_{95\%}=\widehat{\beta}\pm1.96SE \]

Likelihood-Ratio Test

\[ LR=2\left(\ell_{full}-\ell_{reduced}\right) \]
  • Full model과 nested reduced model 비교
  • 동일한 random-effect structure 사용
  • 두 모델 모두 수렴하고 관측 수가 같을 때 유효

BFI Effect Conversion

\[ \mathrm{Ratio}=e^{\widehat{\beta}} \]
\[ \%\Delta=100\left(e^{\widehat{\beta}}-1\right) \]
  • BFI log-ratio coefficient만 변환
  • BVI coefficient에는 적용하지 않음

Likelihood-Ratio Tests

Primary Model Tests

  • Load × SDS: interaction 제거 모델과 비교
  • All load terms: load main effect와 interaction 동시 제거
  • All SDS terms: SDS main effect와 interaction 동시 제거

Secondary Model Tests

  • Load × Processing: interaction 제거 모델과 비교
  • All processing terms: processing 관련 항 동시 제거
  • All load terms: load 관련 항 동시 제거
  • Adjusted SDS: SDS 항 제거 모델과 비교

Random-Effect Summary

Covariance Components

  • Random-intercept variance
  • Random-load-slope variance
  • Intercept–slope covariance
  • Residual variance

ICC at Mean Load

\[ ICC_{L_c=0} = \frac{\sigma_{b0}^{2}} {\sigma_{b0}^{2}+\sigma_{\varepsilon}^{2}} \]
  • ICC: Intraclass Correlation Coefficient
  • Load_c = 0, 즉 load 2.5에서 계산

Interpretation Notes

Model Assumptions

  • Load 1–4를 continuous predictor로 사용하므로 선형 load trend를 가정합니다.
  • 4-back에서 감소하는 inverted-U와 같은 비선형 반응은 직접 모델링하지 않습니다.
  • Wald z 및 chi-square LRT는 asymptotic approximation을 사용합니다.
  • 잔차 정규성, 이분산성 및 영향점 진단은 코드에 포함되어 있지 않습니다.

Design-Specific Considerations

  • 모든 load가 동일 baseline을 공유하므로 baseline error가 여러 Outcome에 공통으로 들어갑니다.
  • RawDarkOnly와 NoiseReduced는 동일 원신호에서 파생된 paired data입니다.
  • 현재 Secondary model은 동일 Subject × SDS × Load pair의 추가 상관을 별도로 모델링하지 않습니다.
  • 여러 fixed effects와 contrasts에 대한 다중검정 보정은 적용하지 않습니다.
