Michelson Interferometer Simulation

This page visualizes a Michelson interferometer as an interactive simulation. Moving the reference or sample mirror changes the optical path difference, which changes the phase difference between the two returning beams. The resulting detector intensity and interference fringe are updated in real time.

The detector intensity is modeled as I = Ir + Is + 2√(IrIs)Vcos(Δφ), where V decreases when the optical path difference exceeds the coherence length. Mirror displacement changes the optical path difference by a round-trip factor of two.
