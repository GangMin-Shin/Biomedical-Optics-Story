Michelson Interferometer Simulation

아래 시뮬레이션은 Michelson Interferometer의 reflective geometry를 바탕으로, beam splitter에서 나뉜 reference arm과 sample arm이 각각 거울에서 반사된 뒤 다시 recombination 되면서 detector에 나타나는 간섭 패턴을 보여줍니다. 여기서 ΔL은 sample arm에 삽입된 sample 또는 phase plate에 의해 생기는 one-way optical path change로 표현했으며, Michelson 구조에서는 빛이 sample arm을 왕복하므로 OPD = 2ΔL로 계산됩니다.

회로 해석: 입력광 U₀는 beam splitter에서 reference arm과 sample arm으로 나뉩니다. reference arm의 빛은 위쪽 reference mirror에서 반사되고, sample arm의 빛은 sample 또는 phase plate를 지난 뒤 sample mirror에서 반사됩니다. 두 빛은 다시 beam splitter에서 recombination 되며, 아래쪽 CCD-like detector에서 간섭 신호로 측정됩니다. Michelson 구조에서는 sample arm을 왕복하므로 sample-induced one-way path change ΔL은 detector에서 OPD = 2ΔL로 반영됩니다.

시뮬레이션 제어

Sample-arm one-way path change, ΔL 0.00 μm
거울 자체를 움직이는 방식이 아니라, sample arm에 삽입된 sample 또는 phase plate 때문에 생기는 one-way optical path change입니다. Michelson 구조에서는 빛이 sample arm을 왕복하므로 detector에서의 optical path difference는 OPD = 2ΔL입니다.
Wavelength, λ 633 nm
같은 OPD라도 λ가 짧을수록 phase difference가 더 크게 변합니다.
Fringe spatial frequency 7.0 cycles
Detector 화면의 줄무늬가 얼마나 촘촘한지를 의미합니다. 값을 올리면 CCD-like screen과 아래 canvas에서 fringe가 더 촘촘해집니다.
Visibility, V 0.90
fringe contrast를 의미합니다. 값을 낮추면 bright/dark 차이가 줄어들고, CCD-like screen과 canvas의 contrast도 함께 약해집니다.
Optical path difference, OPD 0.00 μm
Phase difference, Δφ 0.00 rad
Detector center intensity 0.95
Normalized output power 95.0%
사용한 식
I(x) = I₀ / 2 · [1 + V cos(2πfx + Δφ)]
OPD = 2ΔL
Δφ = 2π · OPD / λ = 4πΔL / λ

Detector 간섭 패턴

위쪽 영역은 CCD-like detector fringe pattern이고, 아래쪽 영역은 같은 phase condition에서의 normalized detector intensity waveform입니다. ΔL, λ, fringe spatial frequency, visibility를 조절하면 detector 출력이 동시에 변합니다.

구성요소 설명

Laser : coherent light source입니다.
BS : 입력광을 reference arm과 sample arm으로 나누고, 반사되어 돌아온 두 beam을 다시 recombination 시키는 beam splitter입니다.
Reference Mirror : 기준 arm의 optical path를 형성하는 거울입니다.
Sample : sample arm의 optical path length를 바꾸는 phase object 또는 phase plate입니다.
Sample Mirror : sample arm의 빛을 다시 beam splitter 방향으로 반사시키는 거울입니다.
Detector : recombination 이후 나타나는 interference signal을 측정하는 CCD-like detector입니다.

이 간섭계로 볼 수 있는 것

  • Phase difference : reference arm과 sample arm 사이의 위상차를 관찰할 수 있습니다.
  • Sample-induced optical path change : sample의 두께 또는 굴절률 변화가 fringe shift로 나타납니다.
  • 미세 변위(displacement) : mirror 또는 sample 위치 변화가 phase 변화로 나타납니다.
  • 굴절률 변화(refractive index change) : sample arm 안의 매질 변화가 optical path length를 바꿉니다.
  • OCT 기본 원리 : low-coherence source와 reference/sample arm 개념을 연결하면 optical coherence tomography의 기본 구조를 이해할 수 있습니다.

슬라이더 변화가 회로에서 어떻게 보이는가

  • ΔL : sample plate의 두께 표현이 바뀌고, detector fringe 위치와 출력 세기가 함께 변합니다.
  • λ : 같은 ΔL에서도 phase shift 민감도가 달라집니다.
  • Fringe spatial frequency : CCD-like screen과 canvas의 줄무늬 간격이 직접 바뀝니다.
  • Visibility, V : detector preview와 canvas에서 bright/dark contrast가 약해지거나 강해집니다.
