Basler acA1300-60gm

Camera Overview

Basler acA1300-60gm은 Basler ace Classic 계열에 포함되는 monochrome area-scan industrial camera입니다. e2v EV76C560 CMOS image sensor를 사용하며, 기본 전체 해상도에서 최대 60 fps의 영상을 GigE 인터페이스로 전송합니다.

  • Area-scan camera: 한 번의 exposure로 2차원 영상을 획득하는 카메라
  • CMOS: Complementary Metal-Oxide-Semiconductor
  • GigE: Gigabit Ethernet 기반 영상 전송 인터페이스
  • Monochrome: Bayer color filter가 없는 흑백 영상 센서 구성

일반형과 NIR 모델의 구분

acA1300-60gm과 acA1300-60gmNIR은 이름이 유사하지만 센서와 spectral response가 서로 다른 모델입니다.

  • acA1300-60gm: EV76C560 sensor, visible 모델
  • acA1300-60gmNIR: EV76C661 sensor, NIR-enhanced 모델
  • 785 nm 또는 808 nm 측정 시스템에서 실제 카메라가 NIR 모델인지 일반형인지 제품 라벨과 serial 정보를 통해 반드시 확인해야 합니다.

What Is EMVA Data?

EMVA의 의미

EMVA는 European Machine Vision Association 의 약자입니다. 유럽을 중심으로 머신비전 산업의 기술 표준화와 산업 협력을 추진하는 조직입니다.

EMVA 1288의 의미

EMVA 1288은 머신비전용 image sensor와 camera의 성능을 측정하고 계산하며 결과를 표시하는 방법을 정의한 표준입니다. 제조사마다 서로 다른 조건과 용어로 제시하던 카메라 성능을 공통 기준으로 비교하기 위해 만들어졌습니다.

EMVA Data가 필요한 이유

  • 카메라의 감도와 noise를 객관적으로 비교할 수 있습니다.
  • 단순 ADC bit depth와 실제 유효 sensitivity를 구분할 수 있습니다.
  • camera output DN을 photoelectron 수로 환산할 근거를 제공합니다.
  • shot noise, dark noise 및 saturation 특성을 정량화할 수 있습니다.
  • 특정 실험에 적합한 exposure와 intensity 범위를 설정할 수 있습니다.

현재 표준 구조

  • Release 4.0 Linear: 선형 반응을 가지며 전처리가 제한된 카메라 평가
  • Release 4.0 General: 비선형 response, 내부 전처리, 렌즈 포함 시스템 및 multimodal sensor를 포함하는 확장 평가
  • Release 4.0은 기존 Release 3.1을 대체하는 공식 체계입니다.

How EMVA 1288 Is Measured

1

Stable Illumination

균일하고 안정적인 광원을 사용하여 sensor plane에 알려진 photon exposure를 제공합니다.

2

Bright Frames

여러 illumination level에서 복수의 illuminated frame을 획득합니다.

3

Dark Frames

광 입력을 차단한 상태에서 dark frame을 획득하여 offset과 temporal dark noise를 측정합니다.

4

Mean and Variance

각 광량에서 평균 DN, temporal variance 및 spatial variation을 계산합니다.

5

Model Fitting

photon transfer model을 이용해 gain, quantum efficiency, noise 및 saturation parameter를 추정합니다.

균일 조명의 중요성

Pixel response의 spatial variation을 측정하려면 센서 전체에 가능한 한 균일한 irradiance가 입사해야 합니다. 조명 자체의 불균일성이 크면 카메라의 PRNU와 광원의 nonuniformity를 구분하기 어려워집니다.

복수 프레임이 필요한 이유

한 장의 영상만으로는 시간에 따라 변하는 temporal noise와 pixel마다 고정된 spatial nonuniformity를 분리할 수 없습니다. 동일 조건에서 여러 frame을 획득해야 frame-to-frame variance와 pixel-to-pixel variation을 각각 계산할 수 있습니다.

기본 광전자 변환 모델

입사 photon은 quantum efficiency에 따라 photoelectron으로 변환되고, camera electronics를 통과한 뒤 digital number로 출력됩니다.

\[ \mu_e = \eta \mu_p \]
\[ \mu_y = K\mu_e+\mu_{y,\mathrm{dark}} \]
  • \(\mu_p\) [photon pixel\(^{-1}\)]: pixel에 입사한 평균 photon 수
  • \(\eta\) [e\(^{-}\) photon\(^{-1}\)]: quantum efficiency
  • \(\mu_e\) [e\(^{-}\)]: 생성된 평균 photoelectron 수
  • \(K\) [DN e\(^{-1}\)]: system gain 또는 conversion gain
  • \(\mu_y\) [DN]: 평균 camera output
  • \(\mu_{y,\mathrm{dark}}\) [DN]: dark offset

Photon Transfer Curve

PTC(Photon Transfer Curve)는 평균 신호와 temporal variance의 관계를 이용하여 camera gain과 noise 특성을 구하는 방법입니다. shot-noise-dominant linear region에서는 variance가 평균 신호에 거의 선형적으로 비례합니다.

\[ \sigma_y^2 \approx K \left( \mu_y-\mu_{y,\mathrm{dark}} \right) + \sigma_{y,\mathrm{dark}}^2 \]
  • \(\sigma_y^2\) [DN²]: illuminated frame의 temporal variance
  • \(K\) [DN e\(^{-1}\)]: variance-mean curve의 linear region 기울기
  • \(\sigma_{y,\mathrm{dark}}\) [DN RMS]: temporal dark noise
  • \(g=K^{-1}\) [e\(^{-}\) DN\(^{-1}\)]: inverse system gain

Main EMVA 1288 Parameters

Quantum Efficiency

입사 photon 중 검출 가능한 photoelectron으로 변환된 비율입니다. 파장에 따라 달라지므로 실제 사용 wavelength에서 확인해야 합니다.

\[ \eta = \frac{\mu_e}{\mu_p} \]
단위: \(\left[\mathrm{e^-\,photon^{-1}}\right]\) 또는 \([\%]\)

System Gain

sensor에서 생성된 electron 수가 camera의 digital output으로 변환되는 비율입니다.

\[ K = \frac{ \mu_y-\mu_{y,\mathrm{dark}} }{ \mu_e } \]
단위: \(\left[\mathrm{DN\,e^{-1}}\right]\)

Inverse System Gain

1 DN이 평균적으로 몇 개의 electron에 해당하는지를 나타냅니다. DN domain과 electron domain 사이의 변환에 사용됩니다.

\[ g = \frac{1}{K} \]
단위: \(\left[\mathrm{e^-\,DN^{-1}}\right]\)

Temporal Dark Noise

광 입력이 없는 상태에서 동일 pixel의 출력이 시간에 따라 변동하는 RMS noise입니다.

\[ \sigma_d = \frac{ \sigma_{y,\mathrm{dark}} }{ K } \]
단위: \(\left[\mathrm{e^-\,RMS}\right]\)

Saturation Capacity

출력이 saturation에 도달하기 전에 pixel이 저장하거나 표현할 수 있는 최대 signal electron 수입니다.

\[ \mu_{e,\mathrm{sat}} = \frac{ \mu_{y,\mathrm{sat}} - \mu_{y,\mathrm{dark}} }{ K } \]
단위: \(\left[\mathrm{e^-}\right]\)

Absolute Sensitivity Threshold

출력 신호의 SNR이 1에 도달하기 위해 필요한 최소 평균 입사 photon 수를 나타냅니다.

\[ \mu_{p,\min} = \frac{ 1+ \sqrt{ 1+ 4\sigma_d^2 } }{ 2\eta } \]
단위: \(\left[\mathrm{photon\,pixel^{-1}}\right]\)

Signal-to-Noise Ratio

평균 signal과 전체 temporal noise의 비율입니다. 광량이 증가하면 일반적으로 증가하지만 saturation 부근에서는 정상적인 선형 관계가 유지되지 않습니다.

\[ \mathrm{SNR} = \frac{ \mu_e }{ \sqrt{ \sigma_d^2+\mu_e } } \]
단위 없음

Dynamic Range

saturation signal과 최소 검출 가능 signal 사이의 범위를 나타냅니다. ADC bit depth와 동일한 개념은 아닙니다.

\[ \mathrm{DR} = 20 \log_{10} \left( \frac{ \mu_{p,\mathrm{sat}} }{ \mu_{p,\min} } \right) \]
단위: \([\mathrm{dB}]\)

DSNU

DSNU는 Dark Signal Non-Uniformity의 약자이며, 균일한 dark condition에서 pixel별 평균 dark offset이 서로 다른 정도입니다.

\[ \mathrm{DSNU} = \frac{ \sigma_{\mu_{y,\mathrm{dark}}} }{ K } \]
단위: \(\left[\mathrm{e^-}\right]\)

PRNU

PRNU는 Photo Response Non-Uniformity의 약자이며, 균일 조명에서 pixel별 광응답 sensitivity가 서로 다른 정도입니다.

\[ \mathrm{PRNU} = \frac{ \sigma_{\mu_y} }{ \mu_y-\mu_{y,\mathrm{dark}} } \times 100 \]
단위: \([\%]\)

Dark Current

광자가 입사하지 않아도 열적 과정 등에 의해 생성되는 electron의 시간당 증가율입니다. Exposure time과 sensor temperature의 영향을 받습니다.

\[ \mu_{I,\mathrm{dark}} = \frac{ \Delta\mu_{e,\mathrm{dark}} }{ \Delta t_{\mathrm{exp}} } \]
단위: \(\left[ \mathrm{ e^-\, pixel^{-1}\, s^{-1} } \right]\)

Linearity

incident exposure와 camera output 사이의 측정 관계가 이상적인 선형 회귀선에서 얼마나 벗어나는지를 나타냅니다.

\[ \delta_{\mathrm{lin}} = \frac{ \mu_y-\mu_{y,\mathrm{fit}} }{ \mu_{y,\mathrm{fit}} - \mu_{y,\mathrm{dark}} } \times 100 \]
단위: \([\%]\)

Terminology

  • ADC: Analog-to-Digital Converter
  • DN: Digital Number
  • LSB: Least Significant Bit
  • RMS: Root Mean Square
  • QE: Quantum Efficiency
  • PTC: Photon Transfer Curve
  • DSNU: Dark Signal Non-Uniformity
  • PRNU: Photo Response Non-Uniformity
  • SNR: Signal-to-Noise Ratio
  • DR: Dynamic Range
