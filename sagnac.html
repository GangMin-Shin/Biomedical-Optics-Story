Sagnac Interferometer Simulation

아래 시뮬레이션은 Sagnac Interferometer의 closed-loop geometry를 바탕으로, 하나의 입력광이 beam splitter에서 clockwise beam과 counter-clockwise beam으로 나뉘어 동일한 closed optical loop를 반대 방향으로 진행한 뒤 다시 recombination 되는 과정을 보여줍니다. loop가 회전하면 두 beam 사이에 Sagnac phase shift가 생기며, 이 위상차가 detector intensity와 fringe shift로 나타납니다.

회로 해석: beam splitter에서 입력광 U₀가 clockwise beam과 counter-clockwise beam으로 나뉩니다. 두 beam은 같은 closed optical loop를 서로 반대 방향으로 진행한 뒤 다시 beam splitter에서 recombination 됩니다. loop가 회전하면 두 beam의 propagation time이 달라져 상대 위상차가 생기고, 이 위상차가 detector의 intensity와 fringe shift로 나타납니다.

시뮬레이션 제어

Rotation rate, Ω 25 rad/s
loop의 angular velocity입니다. 부호가 바뀌면 CW/CCW beam 사이의 phase shift 방향도 반대로 바뀝니다.
Loop area, A 0.80 m²
closed optical loop가 둘러싸는 면적입니다. 면적이 커질수록 같은 Ω에서도 Sagnac phase shift가 커집니다.
Wavelength, λ 785 nm
같은 A와 Ω에서도 λ가 짧을수록 phase shift가 커집니다.
Fringe spatial frequency 7.0 cycles
Detector 화면의 줄무늬가 얼마나 촘촘한지를 의미합니다. 값을 올리면 CCD-like screen과 canvas에서 fringe가 더 촘촘해집니다.
Visibility, V 0.80
interference visibility입니다. 값이 낮아지면 detector fringe contrast가 감소합니다.
Phase offset, φ₀ 0.00 rad
detector operating point를 이동시키는 초기 phase offset입니다.
Simulation gain 1.0 × 10⁶
실험실 크기의 Sagnac phase는 매우 작을 수 있으므로 시각화를 위해 phase response만 증폭합니다. 실제 물리식 자체를 바꾸는 값은 아닙니다.
Physical Sagnac phase 0.000e+0 rad
Display phase 0.00 rad
Detector center intensity 0.90
Normalized output power 90.0%
사용한 식
Δφ = 8πAΩ / λc
I(x) = I₀ / 2 · [1 + V cos(2πfx + Δφdisplay)]
Δφdisplay = gain · Δφ + φ₀

Detector 간섭 패턴

위쪽 영역은 CCD-like detector fringe pattern이고, 아래쪽 영역은 같은 phase condition에서의 normalized detector intensity waveform입니다. 실제 Sagnac phase는 매우 작을 수 있으므로, 화면 표시에는 simulation gain이 적용된 display phase를 사용합니다.

구성요소 설명

Laser : coherent light source입니다.
Beam Splitter : 입력광을 clockwise beam과 counter-clockwise beam으로 나누고, 돌아온 두 beam을 다시 recombination 시킵니다.
M1–M4 : closed optical loop를 형성하는 mirrors입니다.
Closed optical loop : 두 beam이 같은 경로를 반대 방향으로 진행하는 Sagnac geometry의 핵심 구조입니다.
Rotation Ω : loop의 회전으로 인해 CW/CCW beam 사이의 phase shift가 발생합니다.
Detector : recombination 이후 나타나는 interference signal을 측정하는 CCD-like detector입니다.

이 간섭계로 볼 수 있는 것

  • Sagnac phase shift : 회전에 의해 CW/CCW beam 사이에 생기는 상대 위상차를 확인할 수 있습니다.
  • Rotation sensing : detector intensity 변화로 loop의 angular velocity를 감지하는 원리를 볼 수 있습니다.
  • Loop area sensitivity : loop area가 커질수록 phase shift가 증가하는 관계를 확인할 수 있습니다.
  • Wavelength dependence : wavelength가 짧을수록 같은 회전에서 phase sensitivity가 증가합니다.
  • Fringe contrast : visibility가 낮아지면 detector fringe contrast가 감소하는 것을 확인할 수 있습니다.

슬라이더 변화가 회로에서 어떻게 보이는가

  • Ω : display phase가 이동하고 detector intensity와 fringe 위치가 변합니다.
  • A : 같은 Ω에서도 phase shift 크기가 달라집니다.
  • λ : 같은 A와 Ω에서도 phase sensitivity가 바뀝니다.
  • Fringe spatial frequency : CCD-like screen과 canvas의 줄무늬 간격이 직접 바뀝니다.
  • Visibility, V : detector preview와 canvas에서 bright/dark contrast가 약해지거나 강해집니다.
  • Simulation gain : 실제 phase는 그대로 두고, 화면에서 보이는 phase response만 크게 표시합니다.
