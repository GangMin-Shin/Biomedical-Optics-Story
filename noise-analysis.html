Noise Analysis

  • Raw speckle image에서 프레임별 평균 intensity와 spatial variance를 계산하여 \(K_{\mathrm{raw}}^{2}\)를 구합니다.
  • 15 mm와 20 mm SDS 채널에서 shot noise와 dark noise가 제거된 \(K_{\mathrm{corrected}}^{2}\)를 계산합니다.

Image Configuration

Image Region

  • ROI: 21 × 21 pixels
  • Pixels per image: 441 pixels
  • 분석 방식: Spatial speckle contrast

Measurement Channels

  • Ch. 1: 15 mm SDS
  • Ch. 2: 20 mm SDS
  • SDS: Source-Detector Separation

Detector Specifications

  • Camera: Basler acA1300-60gm GigE Camera
  • Sensor: e2v EV76C560 CMOS
  • Resolution: 1282 × 1026 pixels
  • Sensor format: 1/1.8 inch
  • Pixel size: 5.3 × 5.3 μm
  • Maximum frame rate: 60 fps
  • Image type: Monochrome
  • Camera interface: GigE
  • Exposure time used: 2 ms
  • Inverse system gain used: 2.4 e⁻/DN
  • Quantum efficiency used: 0.35
  • Specification dark noise used: 24.7 e⁻
Exposure time, inverse system gain, quantum efficiency 및 specification dark noise는 현재 Python 분석 코드에 입력된 노이즈 계산 파라미터입니다.

Signal Calculation

Mean Intensity
\[ \mu_I = \frac{1}{N} \sum_{i=1}^{N} I_i \]
  • \(I_i\): ROI 내 \(i\)번째 pixel의 intensity
  • \(N = 441\): 한 영상에서 분석되는 전체 pixel 수
  • 단위는 DN(Digital Number)입니다.
Spatial Variance
\[ \sigma_I^2 = \frac{1}{N-1} \sum_{i=1}^{N} \left(I_i-\mu_I\right)^2 \]
  • 한 프레임 내 pixel intensity의 공간적 변동을 나타냅니다.
  • Python 코드에서는 sample variance를 사용하므로 분모가 \(N-1\)입니다.
Speckle Contrast
\[ K_{\mathrm{raw}}^2 = \frac{\sigma_I^2}{\mu_I^2} \]
  • 측정된 전체 spatial variance를 평균 intensity의 제곱으로 정규화한 값입니다.
  • Speckle fluctuation뿐만 아니라 shot noise와 dark noise의 영향도 포함합니다.

Noise Components

Raw Speckle Contrast
\[ K_{\mathrm{raw}}^2 = \frac{\sigma_{\mathrm{raw}}^2}{\mu_I^2} \]
  • \(\sigma_{\mathrm{raw}}^2\)는 raw image에서 측정된 전체 pixel variance입니다.
  • 혈류에 의한 speckle fluctuation과 detector noise가 모두 포함되어 있습니다.
Shot-Noise Contrast
\[ \sigma_{\mathrm{shot}}^2 = \frac{\mu_I}{g} \] \[ K_{\mathrm{shot}}^2 = \frac{\sigma_{\mathrm{shot}}^2}{\mu_I^2} = \frac{1}{g\mu_I} \]
  • \(g = 2.4\ \mathrm{e^- / DN}\)은 Python 코드에 사용된 inverse system gain입니다.
  • Photon arrival의 Poisson fluctuation으로 발생하는 intensity-dependent noise입니다.
  • 평균 intensity가 낮을수록 \(K_{\mathrm{shot}}^2\)의 상대적 영향이 커집니다.
Measured Dark-Noise Contrast
\[ \sigma_{\mathrm{dark,measured}}^2 = \frac{1}{M-1} \sum_{j=1}^{M} \left( D_j-\overline{D} \right)^2 \] \[ K_{\mathrm{dark,measured}}^2 = \frac{ \sigma_{\mathrm{dark,measured}}^2 }{ \mu_I^2 } \]
  • Laser-off background image의 pixel 분산으로부터 직접 측정합니다.
  • Python 코드에서는 128 × 128 background array를 이용하여 dark variance를 계산합니다.
  • \(D_j\)는 background image의 \(j\)번째 pixel 값입니다.
Specification Dark-Noise Contrast
\[ \sigma_{\mathrm{dark,spec}}^2 = \left( \frac{ \sigma_{\mathrm{dark,e^-}} }{ g } \right)^2 \] \[ K_{\mathrm{dark,spec}}^2 = \frac{ \sigma_{\mathrm{dark,spec}}^2 }{ \mu_I^2 } \]
  • 코드에 입력된 dark noise \(24.7\ \mathrm{e^-}\)를 DN 단위로 변환합니다.
  • 측정된 background variance 대신 사양 기반 noise variance를 사용하는 비교 모델입니다.

Speckle Contrast Correction

  • Raw \(K^2\)에서 shot-noise contrast와 dark-noise contrast를 차감합니다.
  • 노이즈 차감 후 variance가 음수가 되는 경우에는 Python 코드와 동일하게 0으로 제한합니다.

Noise-Corrected Speckle Contrast

\[ K_{\mathrm{corrected}}^2 = \max \left( K_{\mathrm{raw}}^2 - K_{\mathrm{shot}}^2 - K_{\mathrm{dark}}^2, \ 0 \right) \]
  • 측정 dark image를 사용할 경우: \(K_{\mathrm{dark}}^2 = K_{\mathrm{dark,measured}}^2\)
  • 사양 기반 dark noise를 사용할 경우: \(K_{\mathrm{dark}}^2 = K_{\mathrm{dark,spec}}^2\)

Variance-Domain Expression

\[ K_{\mathrm{corrected}}^2 = \frac{ \max \left( \sigma_{\mathrm{raw}}^2 - \sigma_{\mathrm{shot}}^2 - \sigma_{\mathrm{dark}}^2, \ 0 \right) }{ \mu_I^2 } \]
  • 실제 Python 코드에서는 먼저 variance 영역에서 noise를 차감한 후 \(\mu_I^2\)로 정규화합니다.
  • 따라서 위 식은 코드에서 계산하는 \(K_{\mathrm{corrected}}^2\)와 동일합니다.
