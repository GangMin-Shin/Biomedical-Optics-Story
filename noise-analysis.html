Noise Analysis

  • CCD에서 측정된 영상에 평균 dark image를 픽셀별로 차감한 영상을 본 분석의 raw image로 사용합니다.
  • Raw \(K^2\)에서 shot noise, temporal dark noise 및 quantization noise 성분을 분리하여 보정된 speckle contrast를 계산합니다.

Signal Calculation

Dark-Subtracted Raw Image
\[ I_{\mathrm{raw}}(x,y,t) = I_{\mathrm{CCD}}(x,y,t) - \overline{D}(x,y) \]
  • \(I_{\mathrm{CCD}}\) [DN]: CCD에서 측정된 pixel intensity
  • \(\overline{D}\) [DN]: 충분한 수의 dark frame을 평균한 pixel-wise dark image
  • \(I_{\mathrm{raw}}\) [DN]: 평균 dark image가 차감된 pixel intensity
  • \(x,y\) [pixel]: ROI 내부의 공간 좌표
  • \(t\) [s]: 영상이 측정된 시간
Mean Intensity
\[ \mu_I = \frac{1}{N} \sum_{i=1}^{N} I_{\mathrm{raw},i} \]
  • \(I_{\mathrm{raw},i}\) [DN]: \(I_{\mathrm{raw}}\)에서 ROI 내 \(i\)번째 pixel intensity
  • \(N\) [pixel]: 전체 pixel 수
  • \(\mu_I\) [DN]: 프레임별 평균 intensity
Spatial Variance
\[ \sigma_I^2 = \frac{1}{N-1} \sum_{i=1}^{N} \left( I_{\mathrm{raw},i}-\mu_I \right)^2 \]
  • \(\sigma_I^2\) [DN²]: 한 프레임 내 pixel intensity의 spatial variance
Speckle Contrast
\[ K_{\mathrm{raw}}^2 = \frac{\sigma_I^2}{\mu_I^2} \]
  • \(K_{\mathrm{raw}}^2\): noise 성분을 포함한 raw squared speckle contrast

Noise Components

Shot-Noise Contrast
\[ K_{\mathrm{shot}}^2 = \frac{ \sigma_{\mathrm{shot}}^2 }{ \mu_I^2 } = \frac{1}{g\mu_I} \]
  • Shot noise : photon-counting Poisson noise
  • \(g\) [e⁻/DN] : inverse system gain, \(2.4\ \mathrm{[e^- / DN]}\)
  • \(\mu_I\) [DN] : 프레임별 평균 pixel intensity
  • \(\sigma_{\mathrm{shot}}^2\) [DN²] : shot-noise variance, \(\sigma_{\mathrm{shot}}^2=\mu_I/g\)
Temporal Dark-Noise Contrast
\[ K_{\mathrm{dark}}^2 = \frac{ \sigma_{\mathrm{dark}}^2 }{ \mu_I^2 } \]
  • Temporal dark noise : exposure 중 detector에서 발생하는 temporal noise
  • \(\sigma_{\mathrm{dark,e^-}}\) [e⁻ RMS] : electron domain의 temporal dark noise, \(24.7\ \mathrm{[e^- RMS]}\)
  • \(\sigma_{\mathrm{dark}}\) [DN RMS] : DN domain의 temporal dark noise, \(\sigma_{\mathrm{dark,e^-}}/g\)
  • \(\sigma_{\mathrm{dark}}^2\) [DN²] : temporal dark-noise variance
Quantization-Noise Contrast
\[ K_q^2 = \frac{ \sigma_q^2 }{ \mu_I^2 } = \frac{ \Delta^2 }{ 12\mu_I^2 } \]
  • Quantization noise : ADC의 discrete DN 변환 오차
  • \(\Delta\) [DN/LSB] : quantization step, \(1\ \mathrm{[DN/LSB]}\)
  • \(\sigma_q\) [DN RMS] : quantization-noise RMS, \(\Delta/\sqrt{12}\)
  • \(\sigma_q^2\) [DN²] : quantization-noise variance, \(\Delta^2/12\)

Speckle Contrast Correction

Noise-Corrected Speckle Contrast

\[ K_f^2 = K_{\mathrm{raw}}^2 - K_{\mathrm{shot}}^2 - K_{\mathrm{dark}}^2 - K_q^2 \]
  • \(K_f^2\): 혈류에 의해 형성된 실제 동적 speckle contrast
  • \(K_{\mathrm{raw}}^2\): 측정된 raw squared speckle contrast
  • \(K_{\mathrm{shot}}^2\): shot-noise contribution
  • \(K_{\mathrm{dark}}^2\): temporal dark-noise contribution
  • \(K_q^2\): ADC quantization-noise contribution
