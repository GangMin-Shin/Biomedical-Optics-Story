Reflection Geometry

This browser simulation visualizes a banana-shaped sensitivity kernel for source-detector pairs placed on the same surface. The Brain model follows a simplified 4-layer head structure, while the Radial model follows a simplified multilayer wrist structure with radial artery ROI.

Simulation Options

This static webpage uses a diffusion-based sensitivity approximation for interactive visualization. The original Python code uses Numba-based Monte Carlo simulation, which cannot run directly on GitHub Pages.
