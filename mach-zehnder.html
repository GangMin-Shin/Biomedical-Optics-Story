Mach-Zehnder Interferometer Simulation

아래 시뮬레이션은 Mach-Zehnder Interferometer의 transmissive geometry를 바탕으로, 두 arm에서 생긴 optical path difference가 두 개의 complementary output에서 어떤 간섭 패턴으로 나타나는지를 보여줍니다. 오른쪽 슬라이더를 조절하면 Detector 1과 Detector 2의 fringe pattern, intensity, contrast가 함께 변합니다.

회로 해석: BS1에서 입력광 U₀가 두 arm으로 분리되고, M1과 M2에서 반사된 뒤 BS2에서 다시 recombination 됩니다. 그 결과 BS2의 오른쪽 출력(D1)과 아래쪽 출력(D2)에서 서로 complementary한 간섭 결과가 나타납니다.

시뮬레이션 제어

Optical path difference, ΔL 0.00 μm
M1을 아주 조금 움직이거나, 한 arm의 굴절률/경로 길이가 바뀐 상황을 단순화해서 표현한 값입니다. ΔL이 변하면 fringe 전체가 좌우로 shift 됩니다.
Wavelength, λ 633 nm
같은 ΔL이라도 λ가 짧을수록 phase difference가 더 크게 변합니다.
Fringe spatial frequency 7.0 cycles
Detector의 줄무늬가 얼마나 촘촘한지를 의미합니다. 값을 올리면 D1, D2 미리보기 화면과 아래 detector canvas에서 fringe가 더 촘촘해집니다.
Visibility, V 0.90
fringe contrast를 의미합니다. 값을 낮추면 bright / dark 차이가 줄어들고, 출력 beam의 강조 효과와 detector preview contrast도 함께 약해집니다.
Phase difference, Δφ 0.00 rad
D1 center intensity 0.95
D1 output power 95.0%
D2 output power 5.0%
사용한 식
I₁(x) = I₀ / 2 · [1 + V cos(2πfx + Δφ)]
I₂(x) = I₀ / 2 · [1 - V cos(2πfx + Δφ)]
Δφ = 2πΔL / λ

Detector 간섭 패턴

위쪽은 D1, 아래쪽은 D2입니다. 두 출력은 complementary output이므로 한쪽이 밝아질수록 다른 한쪽은 상대적으로 어두워집니다.

구성요소 설명

Laser : coherent light source입니다.
BS1 : 입력광을 두 arm으로 분리하는 첫 번째 beam splitter입니다.
M1, M2 : 각 arm의 빛을 반사시켜 BS2로 보내는 mirror입니다.
BS2 : 두 arm의 빛을 다시 recombination 시키는 두 번째 beam splitter입니다.
D1, D2 : recombination 이후 나타나는 두 개의 complementary output을 측정하는 detector입니다.

이 간섭계로 볼 수 있는 것

  • Phase difference : 두 arm 사이의 위상차를 직접적으로 관찰할 수 있습니다.
  • 미세 변위(displacement) : mirror나 sample 위치가 조금만 바뀌어도 fringe가 이동합니다.
  • 굴절률 변화(refractive index change) : sample arm 안의 매질 변화가 optical path length를 바꿉니다.
  • 박막 또는 투명 시료의 두께 변화 : 경로차 변화로 간섭 패턴이 바뀝니다.
  • coherence / alignment 상태 : visibility가 낮아지면 fringe contrast가 줄어들어 정렬이나 coherence 상태를 가늠할 수 있습니다.

슬라이더 변화가 회로에서 어떻게 보이는가

  • ΔL : phase가 바뀌므로 D1, D2의 밝기 분배와 fringe 위치가 바뀝니다.
  • Fringe spatial frequency : D1, D2 detector screen과 canvas의 줄무늬 간격이 직접 바뀝니다.
  • Visibility, V : detector preview와 canvas에서 bright/dark contrast가 약해지거나 강해집니다.
  • λ : 같은 ΔL에서도 fringe shift 민감도가 달라집니다.
