Mach-Zehnder Interferometer Simulation

Photodetector mode: source waveform and D1/D2 complementary temporal signals are displayed.

Insights

  • Phase difference : 두 arm 사이의 optical path difference가 detector output을 변화시킵니다.
  • Eye sample : 눈 모델의 cornea, lens, retina 구조를 sample arm에 배치했습니다.
  • Photodetector : 시간영역 intensity 변화를 oscilloscope 형태로 확인합니다.
  • Spectrometer : 파장별 spectral interference를 확인합니다.
  • Spectrometer + line scan camera : OCT 방식처럼 retina depth profile과 B-scan 유사 출력을 보여줍니다.

Control

Path Difference, ΔL 0.00 μm
Changes the sample-arm optical path delay from 0 to 10 μm.
Wavelength, λ 633 nm
Sets the source wavelength and phase sensitivity.
Fringe Density 7.0 cycles
Adjusts the spectral or spatial modulation density.
Visibility, V 0.90
Adjusts the interference contrast.
Detector Type Photodetector
Phase Difference, Δφ 0.00 rad
D1 Intensity 0.95
D1 Power 95.0%
D2 Power 5.0%

Components

Laser : coherent light source입니다.
BS1 : 입력광을 reference arm과 sample arm으로 나누는 첫 번째 beam splitter입니다.
M1, M2 : 각 arm의 빛을 BS2 방향으로 반사시키는 mirror입니다.
Eye Sample : cornea, lens, retina를 단순화한 sample-arm model입니다.
BS2 : M1 아래에 정렬된 두 번째 beam splitter이며 두 arm을 recombination합니다.
D1, D2 : 선택한 detector model에 따라 temporal, spectral, OCT-like output을 표시합니다.

Effects

  • ΔL : sample arm의 optical path delay가 증가하여 phase shift가 커집니다.
  • λ : 같은 ΔL에서도 wavelength가 짧을수록 phase sensitivity가 증가합니다.
  • Visibility : optical alignment, coherence, polarization mismatch에 의한 contrast 변화를 단순화한 변수입니다.
  • Detector Type : 같은 interferometer라도 검출 방식에 따라 관찰되는 output 형태가 달라집니다.
