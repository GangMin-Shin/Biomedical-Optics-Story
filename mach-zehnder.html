Mach-Zehnder Interferometer Simulation

Upper arm : Laser → BS1 → M1 → BS2 → D1 / D2
Sample arm : Laser → BS1 → M2 → 5-layer Sample → BS2 → D1 / D2
Output : BS2 recombination → complementary interference

Detector Pattern

Top: D1  |  Bottom: D2. The two outputs are complementary.

Insights

  • Phase difference : 두 arm 사이의 위상차를 볼 수 있습니다.
  • Path change : sample의 두께나 굴절률 변화가 fringe shift로 나타납니다.
  • Displacement : 미세 위치 변화가 phase 변화로 연결됩니다.
  • Refractive index : 매질 변화가 optical path length를 바꿉니다.
  • Alignment : visibility 변화로 정렬 상태를 가늠할 수 있습니다.

Control

Fringe Density 7.0 cycles

Components

Laser : coherent light source입니다.
BS1 : 입력광을 transmitted beam과 reflected beam으로 나누는 첫 번째 beam splitter입니다.
M1, M2 : 각 arm의 빛을 BS2 방향으로 반사시키는 mirror입니다.
Sample : sample arm의 optical path length를 바꾸는 5-layer phase object입니다.
BS2 : 두 arm의 빛을 recombination 시키는 두 번째 beam splitter입니다.
D1, D2 : recombination 이후 나타나는 complementary output을 측정하는 detector입니다.

Effects

  • ΔL : sample width와 fringe 위치가 함께 변합니다.
  • Fringe Density : D1, D2 줄무늬 간격이 직접 바뀝니다.
  • Visibility : bright/dark contrast가 약해지거나 강해집니다.
  • λ : 같은 ΔL에서도 phase shift 민감도가 달라집니다.
