Mach-Zehnder Interferometer Simulation

아래 시뮬레이션은 Mach-Zehnder Interferometer의 transmissive geometry를 바탕으로, 두 arm에서 생긴 optical path difference가 두 개의 complementary output에서 어떤 간섭 패턴으로 나타나는지를 보여줍니다. 여기서 ΔL은 M1을 움직이는 방식이 아니라, sample arm에 sample 또는 phase plate가 삽입되어 optical path가 바뀌는 상황으로 표현했습니다.

회로 해석: BS1에서 입력광 U₀가 transmitted beam과 reflected beam으로 나뉩니다. 위쪽 arm은 M1에서 반사되어 BS2로 내려오고, 아래쪽 sample arm은 M2에서 반사된 뒤 sample을 통과하여 BS2로 들어갑니다. BS2에서 두 beam이 recombination 되며, 오른쪽 출력 D1과 아래쪽 출력 D2는 complementary output으로 나타납니다.

시뮬레이션 제어

Sample-induced optical path difference, ΔL 0.00 μm
M1을 움직이는 것이 아니라, 아래쪽 sample arm에 삽입된 sample 또는 phase plate 때문에 생기는 optical path difference입니다. ΔL이 변하면 sample plate의 두께 표현, fringe 위치, D1/D2 출력 세기가 함께 변합니다.
Wavelength, λ 633 nm
같은 ΔL이라도 λ가 짧을수록 phase difference가 더 크게 변합니다.
Fringe spatial frequency 7.0 cycles
Detector 화면의 줄무늬가 얼마나 촘촘한지를 의미합니다. 값을 올리면 D1, D2의 CCD-like screen과 아래 canvas에서 fringe가 더 촘촘해집니다.
Visibility, V 0.90
fringe contrast를 의미합니다. 값을 낮추면 bright/dark 차이가 줄어들고, CCD-like screen과 canvas의 contrast도 함께 약해집니다.
Phase difference, Δφ 0.00 rad
D1 center intensity 0.95
D1 output power 95.0%
D2 output power 5.0%
사용한 식
I₁(x) = I₀ / 2 · [1 + V cos(2πfx + Δφ)]
I₂(x) = I₀ / 2 · [1 − V cos(2πfx + Δφ)]
Δφ = 2πΔL / λ

Detector 간섭 패턴

위쪽은 D1, 아래쪽은 D2입니다. 두 출력은 complementary output이므로 한쪽이 밝아질수록 다른 한쪽은 상대적으로 어두워집니다.

구성요소 설명

Laser : coherent light source입니다.
BS1 : 입력광을 transmitted beam과 reflected beam으로 나누는 첫 번째 beam splitter입니다.
M1, M2 : 각 arm의 빛을 BS2 방향으로 반사시키는 mirror입니다.
Sample : sample arm의 optical path length를 바꾸는 phase object 또는 phase plate입니다.
BS2 : 두 arm의 빛을 recombination 시키는 두 번째 beam splitter입니다.
D1, D2 : recombination 이후 나타나는 두 개의 complementary output을 측정하는 CCD-like detector입니다.

이 간섭계로 볼 수 있는 것

  • Phase difference : 두 arm 사이의 위상차를 관찰할 수 있습니다.
  • Sample-induced optical path change : sample의 두께, 굴절률 변화가 fringe shift로 나타납니다.
  • 미세 변위(displacement) : sample이나 optical element 위치 변화가 phase 변화로 나타납니다.
  • 굴절률 변화(refractive index change) : sample arm 안의 매질 변화가 optical path length를 바꿉니다.
  • coherence / alignment 상태 : visibility가 낮아지면 fringe contrast가 줄어들어 정렬이나 coherence 상태를 가늠할 수 있습니다.

슬라이더 변화가 회로에서 어떻게 보이는가

  • ΔL : sample plate의 두께 표현이 바뀌고, D1/D2 fringe 위치와 출력 세기가 함께 변합니다.
  • Fringe spatial frequency : D1, D2 CCD-like screen과 canvas의 줄무늬 간격이 직접 바뀝니다.
  • Visibility, V : detector preview와 canvas에서 bright/dark contrast가 약해지거나 강해집니다.
  • λ : 같은 ΔL에서도 phase shift 민감도가 달라집니다.
